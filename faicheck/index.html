Descargar Faicheck

Aplicación NO oficial compatible con Windows, Mac OS X y Linux.
Requiere tener Java instalado para funcionar.

Todas tus asignaturas al alcance de tu mano

Olvídate de rastrear archivos nuevos

Sólo tienes que seleccionar una asignatura, marcar los archivos que quieres descargar, y descargarlos. Así de sencillo. ¿No sabes cuáles son los archivos nuevos? El programa los detecta por ti.

Una interfaz cuidada

Con solo lo que necesitas ver

El menú de inicio de sesión cobra protagonismo al arrancar el programa, pidiéndote lo único que necesitas para poder entrar en Faitic.

Y si no quieres escribir tus datos una y otra vez, puedes dejar que el programa los recuerde por ti.

¿En tu ordenador o en un USB?

Elige dónde funcionará el programa

Al ejecutar el programa por primera vez, podrás elegir entre configurarlo solo para tu ordenador, o configurarlo al lado del programa y llevártelo a donde quieras.

Carpetas para cada asignatura

Para tenerlo todo bien ordenado

Podrás elegir una carpeta independiente para cada asignatura, y el contenido se almacenará dentro con la misma estructura que en Faitic.

Finalmente me gustaría hacer una pequeña advertencia. Esta aplicación no es oficial de Faitic. Esto significa que la aplicación no está desarrollada por la Universidad de Vigo, propietaria de la plataforma Faitic, sino por un alumno actual de dicha universidad. Aun así, el servidor usado para acceder a las asignaturas sí es el oficial de Faitic. Para más información sobre el uso de la plataforma Faitic, así como las atribuciones correspondientes, consulta esta página.

Instrucciones de instalación

Simplemente descarga la aplicación y ejecútala. Si todo está correcto, la aplicación se abrirá. Si no, prueba a instalar Java. Si ya estaba Java instalado o la aplicación sigue sin abrirse, consulte la página de errores en la aplicación por si alguien reportó algún error.
